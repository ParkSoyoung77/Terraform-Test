프로젝트 소개

AWS 위에서 동작하는 플랫폼을 가정하고 설계한 하이브리드 클라우드 아키텍처 실습 프로젝트입니다. 1개의 VPC 안에 3개 가용영역에 걸친 퍼블릭 · 프라이빗 · 데이터베이스 서브넷을 구성하여 계층별 보안 경계를 명확히 나눴습니다.

프론트엔드는 S3 정적 웹사이트와 CloudFront를 통해 전 세계에 빠르게 배포되고, 백엔드는 Auto Scaling Group과 ALB로 nginx 인스턴스를 부하분산하며, 데이터는 완전히 격리된 서브넷의 RDS MySQL에 저장됩니다.

1. 네트워크 설계

VPC + 6개 서브넷 + 4개 라우팅 테이블 + IGW/NAT Gateway

2. 컴퓨트 계층

ASG(최소 1, 유지 2, 최대 3) + ALB 단일 진입점

3. 스토리지 & 배포

S3 정적 웹사이트 3종 + CloudFront 배포

4. 데이터 & 서버리스

RDS MySQL(studydb) + Secrets Manager + Lambda + API Gateway
