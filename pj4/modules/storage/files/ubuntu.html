Ubuntu

이 인프라의 웹 서비스 계층에서 사용되는 OS입니다

개요

Ubuntu는 Canonical에서 개발한 Debian 기반의 오픈소스 리눅스 배포판입니다. 안정적인 LTS(Long Term Support) 릴리즈와 방대한 패키지 생태계, apt 패키지 매니저를 통해 서버 환경에서 널리 사용됩니다.

이 프로젝트에서의 역할

본 아키텍처에서는 Auto Scaling Group을 통해 배포되는 EC2 인스턴스의 기본 OS로 Ubuntu를 사용합니다. 인스턴스가 새로 생성될 때 user_data 스크립트가 실행되어 AWS 내부 전용 네트워크를 통해 S3 버킷의 정적 파일을 인스턴스로 동기화한 뒤 웹 서비스를 시작합니다.
