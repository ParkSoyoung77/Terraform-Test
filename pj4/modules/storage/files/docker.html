Docker

컨테이너 기반 애플리케이션 실행 환경

개요

Docker는 애플리케이션과 실행에 필요한 모든 의존성을 하나의 컨테이너 이미지로 패키징하여, 어떤 환경에서도 동일하게 실행할 수 있도록 해주는 컨테이너 플랫폼입니다. 가상 머신보다 가볍고 빠른 시작 속도를 제공하며, 마이크로서비스 아키텍처의 핵심 구성 요소로 널리 사용됩니다.

클라우드 환경에서의 활용

Docker 이미지는 ECR(Elastic Container Registry)에 저장되고, ECS나 EKS와 같은 오케스트레이션 서비스를 통해 배포됩니다. GitOps 파이프라인과 결합하면 코드 변경 사항이 자동으로 빌드, 배포되는 CI/CD 워크플로우를 구성할 수 있습니다.
